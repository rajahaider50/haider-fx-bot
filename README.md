# HAIDER FX BOT

Family Safety & Device Management Platform — Android app for parents to monitor and manage children's devices.

## Tech Stack
- **Platform:** Android (Kotlin)
- **Auth:** Firebase Authentication (FREE)
- **Database:** Firebase Realtime Database (FREE)
- **Notifications:** Firebase Cloud Messaging (FREE)
- **Storage:** Cloudinary (FREE — 25GB)
- **Analytics:** Firebase Analytics + Crashlytics (FREE)

## Features
- Parent-Child device pairing
- Live GPS location tracking
- Geofencing (home/school alerts)
- App usage monitoring & blocking
- Screen time limits
- Bedtime mode / Study mode
- Remote device control (lock, alarm, flashlight)
- Emergency SOS
- Daily/Weekly/Monthly reports
- Automation rules
- Admin panel
